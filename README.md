# DMV Rent and Income Project

This project looks at rent and income patterns across the DMV area using public data and regression analysis.

The notebooks clean and combine rent data with income and demographic data, then explore relationships between rent and local economic factors.

## Repo Structure

|-- data/
|   |-- Zip_zori.csv                         # Zillow rent data by ZIP code
|   `-- dmv_rent_income_dataset.csv          # Combined rent, income, and DMV area data
|-- notebooks/

|   |-- regression_lasso_notebook.ipynb      # Regression and Lasso modeling notebook
|   `-- regression_lasso_notebook_with_visualization.ipynb  # Modeling notebook with visualizations
`-- README.md                               # Project overview and repo structure
